Currency Exchange
Back-end API
The Currency Exchange API is a backend application that facilitates currency exchange and account management. It allows users to create multiple accounts, each assigned a unique currency (e.g., AUD, USD, EUR, GBP). Users can deposit funds, transfer money between accounts, and the application automatically performs currency conversions based on current exchange rates fetched from a third-party API. Key features include:
  • Account Management Open and manage multiple accounts in various currencies.
  • Real-Time Currency Exchange: Fetches updated exchange rates from a third-party API.
  • Deposit and Exchange Operations: Easily deposit money and perform currency exchanges.
  • History of Transactions: Records all deposits and exchanges for user review.
  • Authorization and Security: Implements secure user authentication via JWT.
Technologies Used:
  • Python
  • Flask: A lightweight WSGI web application framework for Python.
  • Marshmallow: For object serialization and deserialization.
  • PostgreSQL: Relational database management system.
  • APScheduler: For scheduling background tasks.
  • Third-Party API: Utilizes the Open Exchange Rates API for currency data.
  • JWT token: For secure authentication.

Ripe Tomato
API Web Server
The Ripe Tomato API Web Server is a backend application designed to manage data related to ripe tomatoes. It provides a RESTful API for performing CRUD (Create, Read, Update, Delete) operations on tomato data and integrates with a PostgreSQL relational database for data storage. Key features include:
  • RESTful API: Interact with tomato data using standard HTTP methods.
  • CRUD Operations: Create, read, update, and delete tomato entries.
  • Authentication: Secure access to the API endpoints using JSON Web Tokens (JWT).
  • Database Integration: Utilizes PostgreSQL for reliable data storage.
  • Background Tasks: Periodic tasks for updating data and maintenance.
Technologies Used:
  • Python
  • Flask: A lightweight WSGI web application framework for Python.
  • Marshmallow: For object serialization and deserialization.
  • PostgreSQL: Relational database management system.
  • JWT token: For secure authentication.
