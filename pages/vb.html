File Transfer Application
The File Transfer Application is a simple client-server application developed in VB.NET that enables users to send and receive files over a network using TCP sockets. Utilizing the Winsock2005DLL for handling network connections and file transfers, this application features a user-friendly graphical interface to streamline the file transfer process.

This project was developed during my internship in Italy, where I gained valuable hands-on experience with network programming and client-server architecture.

The application consists of two main components: the Sender and the Receiver. Users can set up either role to initiate or accept file transfers, making it flexible for various networking scenarios.

Features:
  • Send Files: Users can browse and select files to send to a specified IP address and port.
  • Receive Files: The application can listen for incoming file transfers on a designated port.
  • Progress Bar: Users are provided with a visual indication of the file transfer progress, enhancing user experience.
  • Error Handling: The application includes mechanisms for error handling to ensure robust file transfer processes.
Tech Stack:
  • Main Form (Form1):The main interface of the application, containing controls for sending and receiving files.
  • Winsock Controls: Listens for incoming connections and manages received data and connects to the server for sending data.
Event Handlers:
  • Initializes the server to listen for incoming connections on the specified port.
  • Connects the client to the specified server IP address and port.
  • Opens a file dialog for users to select files for sending.
  • Sends the selected file to the connected server.
  • Accepts incoming connection requests from clients.
  • Processes incoming data and files.
  • Updates the progress bar to reflect the current transfer status.
